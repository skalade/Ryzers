### MolmoAct

This directory contains the docker configuration files to run [MolmoAct from
Allen Institute for AI](https://huggingface.co/collections/allenai/molmoact).
The container would run an inference example using the
[MolmoAct-7B-D-0812](huggingface.co/allenai/MolmoAct-7B-D-081) model
checkpoint. The Ryzers example in this repository is adapted from the example
provided by the MolmoAct team.

### Build and run the Docker Image

```sh
ryzers build molmoact
ryzers run
```
